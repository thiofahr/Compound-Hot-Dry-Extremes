# Compound Hot-Dry Events in Indonesia
This repository contains the code and workflows used to analyze historical and future changes in Compound Hot-Dry Events (CHDE) across Indonesia using CMIP6 climate model projections. The analysis includes event detection, frequency and intensity assessment, time of emergence (ToE) estimation, model agreement evaluation, and population exposure assessments under multiple socioeconomic pathways.

The repository is designed to provide a reproducible framework for climate risk assessment and adaptation planning, integrating climate projections, geospatial processing, statistical analysis, and visualization tools.

## Data Sources
- [Daily Downscaled CMIP6](https://nex-gddp-cmip6.s3.us-west-2.amazonaws.com/index.html)
- ERA5
- Population projection datasets

## Study Area
Indonesia
